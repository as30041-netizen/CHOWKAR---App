# SQL Scripts

This folder contains historical and maintenance SQL scripts.

## Active Scripts (in root)
- `CREATE_ALL_RPC_FUNCTIONS.sql` - Main RPC functions (run after schema changes)
- `COMPLETE_NOTIFICATION_TRIGGERS.sql` - Notification triggers

## Archived Scripts (in this folder)
These scripts were used during development and are kept for reference:
- `FIX_*.sql` - Various bug fixes
- `PHASE*.sql` - Feature implementation phases
- `SECURITY_*.sql` - Security hardening scripts

## Usage
To run a script, copy its contents to the Supabase SQL Editor and execute.
